# Currency-converter-using-python
Currency Converter built using Python that fetches real-time exchange rates to perform accurate currency conversions. Designed with a clean logic structure, efficient error handling, and a focus on reliability, making it suitable for real-world financial applications.

A Python-based Currency Converter application with a clear separation of concerns, where `main.py` handles the backend logic and API integration, and `gui.py` provides a user-friendly graphical interface. The application delivers accurate, real-time currency conversions.
